HTML Boilerplate

The steps on HTML files

  1. DocType declaration
    • Contains information such as the document type and language
  2. HTML Element
    • Contains html tag
    • All the webpage goes within this
    • The following items are contained:
    1. Head Element
      • Not usually visible
      • Contains metadata
      • And contains title
    2. Body Element
      • This is usually the visible part
      • It's where most of the processing takes part
